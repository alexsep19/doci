находясь в бранче или master

обновить master в локале
<m>git fetch origin master

догнать бранч до master
<dev>git pull --rebase origin [master]
или 
<dev>git rebase master
<dev>git rebase --abort отмена
разрешение конфликта:
<dev>git add %filename%
git rebase --continue
git rebase --skip изменений в рабочей копии нет.

git push origin Бранч

схлопнуть commit
git reset --soft head~5   включая 5-й
git push origin [branch] --force
git push
----
git checkout [branch] - не обновляет измененные файлы
git reset --hard [branch] - все обновить
